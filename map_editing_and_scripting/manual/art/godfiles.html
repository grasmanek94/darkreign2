Creating GOD Files

Intial Creation

At the top of the FileList dialog type in the desired scale for the new model; press Enter.
Select/Unselect the MrmGen button depending on whether or not mrming is desired on the model.
In general, make sure the HardStrip button is selected and the Multi, Cycle, and Attach buttons are unselected.
Find the base animation cycle xsi file in the list and double click it to load the file.
If there are other animation cycles to load, select the Cycle button then double click on all the cycle files to load.
Switch to the Options dialog.
If a shadow isn't desired, unselect the Shadow button.  If it is, set the shadow radius.
Set the MRM Factor higher than 1 to increase mrming sensitivity, lower than 1 to decrease it.
Set the MRM MinVerts to some reasonable lower limit.
Set the MRM MaxVerts (usually the full max).
One at a time, select each animation cycle in the Animation Cycles list box.
Set each cycle's AnimType (loop, 1way, 2way), AnimSpeed, and if its a move cycle,  MoveSpeed.
MoveSpeed is set by matching the model's animation to the movement of the terrain, eliminating 'skating'.
The AnimRate slider can be used to adjust AnimSpeed and MoveSpeed together so that its easier to make the match.
Select the QuickLight button if non-shaded optimized lighting is desired.
Select the EnvMap button if the model should use environment mapping (greyed if unsupported by the model).
Select/Unselect the Chunkify button depending if auto chunks should be generated.
Type in the desired name for the god file in the FileName edit box.
Click the Save button.

Tweeking GOD Files

Each time a god file is saved a gfg file is produced that maintains all the settings for you.
To re-save a god file click the xsi/god button on the FileList dialog until it displays a list of gfg files.
Double click a gfg file; the viewer tool will reload all the appropriate xsi files with the last settings saved.
Re-adjust the settings and click Save when done.

Creating GOD Files from DR2's cfg files

Just like with gfg files, models can be loaded and configured directly from the appropriate DR2 cfg file.
Click the xsi/god button on the FileList dialog until it displays a  list of cfg files.
Double click a cfg file; the viewer will load the model with the settings it has in DR2.
Adjust settings and click Save when done.
